Transpilando um arquivo TypeScript em JavaScript



Neste tutorial, vamos aprender como transpilar (converter) um arquivo TypeScript (TS) em JavaScript (JS). TypeScript é um superset de JavaScript que adiciona tipagem estática e recursos avançados à linguagem, e a transpilação é o processo de converter código TypeScript em JavaScript para que ele possa ser executado em qualquer ambiente que suporte JS.



Passo 1: Instalação do Node.js



O Node.js é uma plataforma que permite executar código JavaScript fora do navegador, e é necessário para executar arquivos JavaScript em um ambiente de linha de comando. Se você ainda não possui o Node.js instalado, siga as instruções para a instalação adequada do Node.js no seu sistema operacional.

Você pode fazer o download do Node.js no site oficial: https://nodejs.org

Após concluir a instalação do Node.js, verifique se o Node.js e o npm (Node Package Manager) estão corretamente instalados executando os seguintes comandos no terminal ou prompt de comando:

node -v
npm -v

Passo 2: Instalação do TypeScript



Antes de começarmos, é necessário ter o TypeScript instalado em nosso ambiente. Se você ainda não possui o TypeScript instalado, pode fazê-lo através do gerenciador de pacotes npm (Node Package Manager).

Abra o terminal ou prompt de comando e digite o seguinte comando para instalar o TypeScript globalmente:

npm install -g typescript

Passo 3: Criando o arquivo TypeScript



Agora, vamos criar um arquivo TypeScript.

Crie um arquivo chamado app.ts em qualquer diretório de sua preferência.

function greet(name) {
    return `Olá, ${name}!`;
     }
 const message = greet("João");
 console.log(message);

Note que o parâmetro "name" não possui tipagem estabelecida, logo, é atribuido por default o tipo: any



Passo 4: Transpilando o arquivo



Após criar o arquivo TypeScript, estamos prontos para transpilá-lo para JavaScript. No terminal, navegue até o diretório onde você salvou o arquivo app.ts e digite o seguinte comando: 

tsc app.ts

O comando tsc é o compilador TypeScript, e ele irá transpilar o arquivo app.ts para JavaScript, gerando um arquivo chamado app.js.


Passo 5: Verificando o resultado



Após a transpilação ser concluída, você encontrará o arquivo JavaScript resultante (app.js) no mesmo diretório em que o arquivo TypeScript estava.

Agora, você pode executar o arquivo JavaScript normalmente como faria com qualquer outro código JS: 

node app.js

E pronto! Agora você tem um arquivo TypeScript convertido para JavaScript e está pronto para ser executado em qualquer ambiente JavaScript. Lembre-se de que, durante a transpilação, as anotações de tipo presentes no código TypeScript são removidas, pois JavaScript não possui o conceito de tipagem estática do TypeScript.
